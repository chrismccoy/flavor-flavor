# Flavor Flavor

A WordPress blog theme with Bold typography, hard shadows, pop-colour accents, and hover animations.

## Features

- 🎨 Brutalism design system with hard shadows, thick outlines, and pop colours (cyan, yellow, pink)
- 🖋️ IBM Plex Mono + Inter font pairing for bold, readable typography
- 🌊 CSS hover animations on cards and buttons
- 📱 Fully responsive layout with mobile-first Tailwind CSS
- 🧩 14 custom widgets — Search, Categories, Popular Posts, Recent Posts, Archives, Tag Cloud, Calendar, Pages, Recent Comments, Quick Links, Newsletter, Text, Social Media, and About Me
- 🖼️ Custom gallery shortcode output with brutalism-styled grid and zero-dependency lightbox
- 💬 Threaded comments with custom Walker rendering avatar frames, author badges, and accent-colour rotation
- 🔢 Dual pagination styles — numbered pages or newer/older buttons — switchable via the Customizer
- 🎛️ Customizer panels for CTA section, header buttons, footer text, and pagination style
- 📰 Reading time estimates and post view counters
- 🔍 Pretty search URLs (`/search/term/`) with brutalism-styled search forms
- 🎬 Responsive video shortcode with inline-width stripping
- 📸 Featured image support with custom hero and card image sizes
- 🏷️ Category and tag archive templates with accent-coloured badges
